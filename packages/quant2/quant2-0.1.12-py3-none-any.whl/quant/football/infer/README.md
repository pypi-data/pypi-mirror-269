## P-C-R

预测成功率`p`与赔率`c`和收益率`r`关系：

```python
p, c, r = 0.6, 1.7, 1.1

# p * c - (1 - p) * 1 >= r
# p * (c + 1) >= r + 1
# c >= (r + 1) / p - 1

c >= (r + 1) / p - 1
```
